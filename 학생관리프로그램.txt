	//생성자
	자유롭게 이름,나이, 성별 먼저 입력받던지 나중에 입력 받던지
	기본생성자

	// 메서드 (기능) -필요하다고 생각하는 기능 넣어주기
	getter / setter 메서드

	menu() 메서드
		1. 학생 목록 -> information() 실행
		2. 학생 추가 -> add()
		3. 학생 수정 -> modify()
		4. 학생 삭제 -> delete()
		5. 종료
	1번 누르면 information()
		학생 정보보기 (표현 방식은 자유롭게)
	2번 누르면 add()
		학생 추가 (학생 정보 추가 , 성적 입력 등등)
	3번 modify()
		학생 수정 (기존의 학생의 정보가 들어가 있는 경우에만 수정
			나이, 이름 등 선택해서 수정)
	4번 delete()
		학생 삭제 (학생의 정보를 없애는 것, 값들을 null 값으로 바꾼다거나..)
